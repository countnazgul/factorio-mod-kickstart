# Kickstart (Factorio MOD)

Small Factorio mod that kickstart the game by auto research some technologies and gives the player initial set of items to work with.

## Usage

Once new game is started open the console (press `` ` ``) and type `/kickstart`. Once the command is complete your inventory will be populated with items and some technologies will be researched.

## List of items and technologies

- Logistics

- Military
- Radar
- Railway
- Repair pack
- Solar energy
- Steel axe
- Steam power

## Mod the mod

If you want to have other items or technologies just open the mod's `config.lua` in text editor and update the technologies and items list there. The mod location is usually `C:\Users\[USERNAME]\AppData\Roaming\Factorio\mods\kickstart_X.X.X\`

Full list of items and technologies (and a lot of other stuff) can be found at [Factorio wiki page](https://wiki.factorio.com/Data.raw)
